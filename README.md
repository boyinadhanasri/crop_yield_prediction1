# Crop Yield Prediction System

This project predicts crop yield using machine learning based on:
- Weather conditions
- Soil properties
- Historical agricultural data

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Random Forest Algorithm

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the program:
   python main.py
3. Enter input values when prompted
